気象庁データ可視化
==================

# 背景・目的
私は現在東京に住んでいるが，これまでは22年間熊本に住んでいた．
東京の人イメージではどうやら九州は暖かいらしい．
でも実際そんなことは全然ないと思う，
雪もふるし暖房機器に本気出してない分寒いはず．
なので，データを使って「九州も寒いよ！！」ということを証明したい．


# 使用するデータ
熊本・東京の一日ごとの
+ 平均気温
+ 平均湿度

期間：2010-01-01 - 2014-12-31


# 方針
5年分横に並べるか or 一年分に区切ってグラフを重ねるか
重ねるの面倒だし，とりあえず横に並べてみる
